# Electric Vehicle Sales Analysis Dashboard using Power BI

## Project Overview

This project focuses on analyzing Electric Vehicle (EV) sales trends across India using Power BI. The dashboard provides valuable insights into state-wise EV adoption, penetration rates, top-performing manufacturers, vehicle category distribution, and growth trends over time.

The objective of this project is to transform raw EV sales data into meaningful business insights that help in understanding market performance and support strategic decision-making.

---

## Key Features

* Interactive Power BI dashboard for EV sales analysis
* State-wise Electric Vehicle adoption analysis
* EV penetration rate calculation
* Top and Bottom performing states identification
* Top EV manufacturers analysis
* CAGR (Compound Annual Growth Rate) calculation
* Vehicle category distribution (2-Wheelers vs 4-Wheelers)
* Monthly, Quarterly, and Yearly trend analysis
* Dynamic filters and slicers for better user interaction

## Business Insights

* Maharashtra, Karnataka, and Delhi show the highest EV adoption rates
* 2-Wheelers contribute more than 90% of total EV sales
* Government incentives and charging infrastructure are major growth drivers
* Metro cities have stronger EV penetration compared to Tier-2 cities
* Smaller states still require better charging infrastructure support
